Kontrolltöö №1 -- Windowsi käsurea keel

Чтобы скрипт работал корректно, необходимо в «Командной строке» переключить кодировку на UTF-8. Сделать это можно, например, введя в приложении «Выполнить» (Win + R) следующую команду:  cmd /K chcp 65001. Можно просто в «Командной строке» ввести команду chcp 65001.
